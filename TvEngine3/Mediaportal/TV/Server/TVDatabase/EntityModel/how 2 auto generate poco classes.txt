each time you make modifications to the DB u will have to generate new poco classes.

procedure is:

1) open the Model.edmx file.
2) right click anywhere on the visual DB model tool, and select "add code generation item".
3) now select "ado.net self tracking entity generator" - u might have to download a template for this for vs2010.
4) choose the name model, overwrite any files present.
5) now move the folders "Model.tt" and "Model.Context.tt" into project => "Entities".
6) fix up namespaces.. do a ctrl-h (replace) on project entities.
change from:
namespace Mediaportal.TV.Server.TVDatabase.EntityModel
to:
namespace Mediaportal.TV.Server.TVDatabase.Entities
leave the quotes out :)

happy compiling...
